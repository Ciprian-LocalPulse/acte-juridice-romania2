**Cea mai completa colectie de modele gratuite de documente juridice romanesti.**
480 de modele in 51 de categorii, fiecare cu temei legal explicit si checklist de
verificare.

## Ce este si ce nu este

Acest proiect este o colectie educationala de modele. NU este consultanta juridica.
Pentru situatii complexe, consultati intotdeauna un avocat, notar sau institutia
competenta.

## Cum gasesti documentul potrivit

1. **Harta situatiilor** - [docs/harta-situatii.md](docs/harta-situatii.md) - porneste de la situatia ta, nu de la categorie.
2. **Registrul complet** - [templates/INDEX.md](templates/INDEX.md) - toate cele 480 de modele listate.
3. **Cautare GitHub** - foloseste bara de cautare din repo pentru cuvinte cheie.

Acest proiect a fost creat din dorința de a **elimina barierele birocratice** și de a oferi acces gratuit la documente juridice corecte, actualizate și ușor de utilizat.
