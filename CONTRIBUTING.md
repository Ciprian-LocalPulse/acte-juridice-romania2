# Contributing

Multumim pentru ajutor. Proiectul accepta contributii care imbunatatesc claritatea, corectitudinea juridica sau acoperirea situatiilor de viata.

## Flux recomandat

1. Deschide un issue cu modelul vizat.
2. Verifica temeiul legal pe o sursa oficiala.
3. Pastreaza formatul standard si campurile cu paranteze mari.
4. Ruleaza `python scripts/validate_structure.py` si `python scripts/check_links.py`.
5. Trimite un pull request cu descriere scurta si sursa verificata.
