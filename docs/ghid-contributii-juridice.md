# Ghid contributii juridice

Contributiile trebuie sa pastreze structura standard a fiecarui model: `Cand se foloseste`, `Temei legal`, `Model`, `Checklist inainte de semnare`.

## Reguli

- foloseste numai surse oficiale pentru temei legal;
- nu introduce diacritice in fisierele de model;
- marcheaza toate campurile editabile cu paranteze mari;
- explica intr-un pull request ce articol sau institutie a fost verificata.
