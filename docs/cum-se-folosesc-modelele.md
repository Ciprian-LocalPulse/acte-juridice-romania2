# Cum se folosesc modelele

1. Alege modelul din `templates/INDEX.md` sau din `docs/harta-situatii.md`.
2. Completeaza doar campurile marcate cu paranteze mari, de tipul `[NUME_COMPLET]`.
3. Pastreaza temeiul legal si checklistul in documentul final.
4. Anexeaza copii lizibile ale actelor mentionate.
5. Pentru acte cu forma speciala, mergi la notar, avocat sau institutia competenta.
